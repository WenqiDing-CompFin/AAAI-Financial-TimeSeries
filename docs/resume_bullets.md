# Resume Wording

## Approved description

- Built a reproducible financial forecasting baseline and time-series research
  scaffold with chronological train, validation, and held-out test partitions,
  validation-only Ridge model selection, leakage-controlled preprocessing, and
  automated tests.
- Established persistence and regularized linear baselines for a future
  event-context modeling study, reporting mixed held-out error and directional
  results rather than presenting synthetic metrics as market evidence.

## Claim boundary

Use the phrases **forecasting baseline** or **research scaffold**. Do not use
"complete TimeCAP reimplementation," "TimeCAP model reproduction," or any claim
of S&P 500 minute-level performance. This repository does not yet implement the
paper's LLM-agent contextualization and augmentation architecture.

The primary application project link remains `quant-factor-research`; this
repository is supporting evidence of time-series evaluation discipline.
